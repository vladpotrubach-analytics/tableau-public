# CO2 Per Capita World Map

**Tool:** Tableau Public  
**Dataset:** CO2 emissions by country  
**Metric:** CO2 emissions per capita (metric tons)

## Description
Interactive world map showing CO2 emissions per capita by country.
The visualization allows filtering by year to explore how emissions change over time.

## Business / analytical questions
- Which countries have the highest CO2 emissions per capita?
- How does CO2 per capita differ between regions?
- How did emissions change over time for selected countries?
## Key insight
High-income and industrialized countries tend to have significantly higher CO2 emissions per capita compared to developing regions.
